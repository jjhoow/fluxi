O que é MCP?

Model Context Protocol (MCP) é um protocolo padronizado que permite conectar seu agente a servidores externos que expõem ferramentas (tools) especializadas.

1 Conecte servidores: STDIO (comando local) ou HTTP (servidor remoto)
2 Sincronize automaticamente: As tools do servidor ficam disponíveis para o agente
3 LLM decide: O modelo escolhe quando usar cada tool baseado no contexto
